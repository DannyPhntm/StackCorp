# Font Files

Upload font files here only if the license permits web/public use.

## Rules

- Only upload fonts you have confirmed licensing for (website/public embedding, not just desktop use).
- Do not upload paid or proprietary fonts unless licensing has been explicitly confirmed by Daniyal.
- Recommended format: `.woff2` (smallest, best browser support). Include `.woff` as a fallback only if needed.
- Name files clearly by family and weight, e.g. `inter-regular.woff2`, `inter-bold.woff2`.

## Notes

- No fonts exist in this repo yet — this folder is a placeholder until real, licensed font files are provided.
- If using a Google Fonts (or similarly freely licensed) font, self-hosting the `.woff2` files here is preferred over a third-party CDN request for performance/privacy, but confirm with Daniyal before adding.
