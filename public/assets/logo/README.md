# Logo Assets

Upload the approved StackCorp logo/icon files here.

## Recommended filenames

- `stackcorp-icon.png` — square icon/mark, transparent background
- `stackcorp-logo.png` — full logo (icon + wordmark combined)
- `stackcorp-wordmark.png` — text-only wordmark
- `stackcorp-icon.svg` — vector icon, if available (preferred for scaling/favicons)

## Notes

- Use transparent backgrounds (PNG or SVG) where possible.
- Do not upload draft, watermarked, or unapproved versions.
- If multiple color variants exist (light/dark), name them clearly, e.g. `stackcorp-logo-dark.png`, `stackcorp-logo-light.png`.
- No logo files exist in this repo yet — this folder is a placeholder until real assets are provided.
