# Malir Cantt Bazaar — Reference Overview

Malir Cantt Bazaar is **StackCorp's** first live product and agency case study.

It is a local marketplace and shops directory for Malir Cantt residents and businesses.

## Purpose

The platform helps people in Malir Cantt:
- buy and sell locally
- discover nearby businesses
- post personal listings
- apply as verified businesses
- create shop profiles
- contact sellers through phone/WhatsApp

## Why It Matters

This project proves we can build and launch a real production system, not just static websites.

It includes:
- user authentication
- email verification
- listings
- image uploads
- Cloudinary integration
- admin dashboard
- business verification
- shops directory
- listing limits
- featured slots
- contact form
- security hardening
- production deployment

## Stack

- Frontend: React / Vite
- Backend: Node / Express
- Database: Neon Postgres
- ORM: Prisma
- Hosting: Vercel + Railway
- Images: Cloudinary
- Email: Resend
- Domain: malircanttbazaar.com
