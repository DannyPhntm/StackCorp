## Monthly Checks

- [ ] Core user flows work end to end
- [ ] Admin/dashboard works

## Backups / Data Checks
- [ ] Backups are running and recent
- [ ] A backup restore is possible (spot-check)
- [ ] Database health looks normal (size, growth, no obvious corruption)
- [ ] No destructive production data changes made without approval

## Broken Links
- [ ] Internal links work
- [ ] External links work
- [ ] No broken images or missing assets

## Forms / Contact Tests
- [ ] Contact / lead forms submit successfully
- [ ] Confirmation and notification emails arrive
- [ ] Spam / rate-limit protection working

## Security Updates
- [ ] Dependencies reviewed; safe updates applied
- [ ] No secrets exposed in repo or logs
- [ ] Auth, roles, and admin guards still enforced
- [ ] Private/admin-only data not exposed publicly
- [ ] HTTPS valid; certificates not expiring soon
- (Follow [SECURITY.md](../SECURITY.md))

## Analytics / SEO Checks
- [ ] Analytics still tracking
- [ ] Key pages indexed; no new crawl errors
- [ ] Titles/meta intact on important pages

## Performance Checks
- [ ] Page load times acceptable (mobile + desktop)
- [ ] No obvious slow pages or large unoptimized assets

## Content Updates
- [ ] Requested content changes applied
- [ ] Outdated info corrected (hours, prices, offers)

## Bug Fixes
- [ ] Reported issues triaged
- [ ] Small in-scope fixes done
- [ ] Larger items logged and quoted separately if out of scope

## Client Reporting
- [ ] Summary sent: what was checked, what changed, what needs attention
- [ ] Any recommendations or risks flagged
- [ ] Next actions and any client decisions noted
